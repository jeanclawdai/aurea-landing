# CRITICAL-CONVERSATIONS.md

_Protocol for logging important conversations._

## What to Log IMMEDIATELY

Log right after receiving, don't wait:
- Business decisions
- Technical specifications
- Strategic choices
- Personal context about JB or team
- Any commitments made
- Project milestones

## How to Log

1. Write to today's memory file immediately
2. Format: Brief summary + key points + decisions made
3. Then acknowledge: "✅ Logged to memory/YYYY-MM-DD.md"

## Example

```
[IMPORTANT] Discussion about Viraly pricing
- Decision: $29/month for base, $99 for premium
- Action: Update Notion pricing page
- Next: Review with Darwin
```
